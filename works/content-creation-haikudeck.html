Content Creation on SlideShare

Having established itslef as a dominant content distribution platform (especially presentations), SlideShare management started exploring presentation creation opportunities more seriously. SlideShare saw HaikuDeck as an ideal choice to enable its users create content. HaikuDeck is a lightweight presentation creation tool which makes storytelling simple, beautiful, and fun. The SlideShare-HaikuDeck partnership was seen as a win-win for both the companies. SlideShare would benefit from a readymade content creation software and HaikuDeck would leverage from large scale exposure from SlideShare.

My role in the project was to design end to end user flows starting from a landing page, a permission page requesting access to user's data, HaikuDeck app interface page to a fully responsive email campaign working with the marketing team.

Content Creation Landing Page

I began with couple of wireframes to address the following objective:

  • to introduce the creation feature on SlideShare in partnership with HaikuDeck
  • to explain the content creation process
  • to show some examples of presentations created with HaikuDeck on SlideShare
  • to inspire users to start creating own stuff

The wireframe was designed to cater to future possibilities of adding similar content creation apps like HaikuDeck on SlideShare. Later on it was agreed that we'd design mockups with only HaikuDeck as an option leaving future venture possibilities aside for the time being.

The final prototype looks quite different from the initial design direction shown in the wireframe resulting out of numerous iterations that were done on the page. You can click on the mockup below to view the live page studded with CSS animations and effects in action.
